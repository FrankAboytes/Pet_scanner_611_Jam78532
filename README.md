# 🐾 Pet Scanner

Sistema de escaneo y gestión de mascotas con código QR. Backend NestJS + TypeScript.

## 🎯 Funcionalidad

- Registro de mascotas con generación de QR
- Escaneo QR para identificar mascotas
- Tracking de ubicación
- Historial y gestión

## 🛠️ Stack

- **Backend:** NestJS + TypeScript
- **Base de datos:** PostgreSQL + TypeORM
- **QR:** Generación automática por mascota

## 🚀 Quick Start

```bash
git clone https://github.com/FrankAboytes/Pet_scanner_611_Jam78532.git
cd Pet_scanner_611_Jam78532
npm install
npm run start:dev
```
